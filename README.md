# The Influence of Family Environment on Academic Performance

## Project Overview
This research project analyzes the impact of family environment factors on student academic performance using multiple linear regression in R.

## Objective
To examine how guardian education, family relationship, family size, parental status, and family support influence final academic grades.

## Methods
- Multiple Linear Regression
- Data Visualization (ggplot2)
- Statistical Analysis in R

## 📁 Dataset
High School Student Performance & Demographics Dataset (Kaggle)

## Key Findings
- Guardian education (higher education) significantly impacts academic performance.
- Other family variables showed limited statistical significance.
- Model R² ≈ 6.8%, indicating additional factors influence performance.

##  Tools Used
- R
- ggplot2
- Multiple Linear Regression
